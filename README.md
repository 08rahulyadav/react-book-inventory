# React Book Inventory App

A React-based Book Inventory application built using Vite and Tailwind CSS.

## Features
- Add new books
- View list of books
- View book details
- Edit existing books
- Delete books
- Form validation
- Scrollable table

## Tech Stack
- React (Vite)
- Tailwind CSS
- React Router
- Axios
- JSON Server (Mock API)

## How to Run Locally
1. Install dependencies  
   npm install

2. Start JSON Server  
   npx json-server --watch db.json --port 5000

3. Start React app  
   npm run dev
